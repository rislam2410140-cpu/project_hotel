CUSTOMIZATION GUIDE
================================================================================

CHANGE PRIMARY COLOR:
Edit /assets/style.css, :root section:
--primary: #YOUR_COLOR;

ADJUST ANIMATION SPEED:
--transition-base: 0.2s; /* was 0.3s */

MODIFY BUTTON PADDING:
.btn { padding: 1rem 2rem; } /* was 0.75rem 1.5rem */

CHANGE SHADOW INTENSITY:
--shadow-lg: 0 20px 40px rgba(0, 0, 0, 0.2);

CUSTOMIZE BORDER RADIUS:
.btn { border-radius: 1rem; } /* was 0.5rem */

All changes are CSS-based and can be done without touching any PHP/HTML.
